# Introducción
En su obra El software toma el mando, Lev Manovich argumenta que hemos pasado de una era de "multimedia" a una hibridación". Mientras que la multimedia consistía en insertar diferentes tipos de medios en un mismo espacio, en la hibridación todas las técnicas se superponen y fusionan para crear una nueva estructura.

A continuación, se analizan dos casos claros de hibridación con técnicas algorítmicas para generar experiencias. Estos ejemplos ha sido seleccionados con el fin de entender la hibridación de una manera sencilla
# Mario Kart Live: Home Circuit
Mario Kart Live: Home Circuit es un juego para Nintendo Switch con fecha de salida del 16 de octubre de 2020.
Este juego es el ejemplo de hibridación perfecta, ya que no hablamos solo de una simple simulación de una carrera, si no de una hibridación entre el espacio tangible (nuestra casa, por ejemplo) y la lógica del software de videojuegos.
Analizándolo con las "gafas de Manovich" podremos ver:
- Hibridación de Interfaces.
  El coche tiene montada una cámara y en la que, gracias al software, la Interfaz de Usuario (UI) se superpone al video en tiempo real. El directo de la cámara se hibrida con el diseño gráfico vectorial de los elementos del juego (los otros corredores, los obstáculos, los premios).
- Fusión de propiedades.
Cuando el jugador recoge un turbo, a través de los vectoriales comentados anteriormente, el motor del coche de juguete acelera físicamente. O, en el caso contrario, si el jugador se choca con, por ejemplo, una pata de una mesa, el personaje digital propio se detiene. Estos dos ejemplos muestran una remixabilidad profunda entre lo real y el código, demostrando que las propiedades del software afectan a las leyes físicas.
- Nueva Gestalt espacial.
El salón deja de ser una simple estancia de la casa para convertise en una base de datos espacial que el software interpreta. No miramos una represetntación del mundo, habitamos en una hibridación inseparable
# Phasmophobia
Phasmophobia es un juego para Playstation 5, Nintendo Switch 2, PC, Xbox Serie X y Series S lanzado el 4 de septiembre de 2020.
Si Manovich analizaba cómo el software hibridaba la imagen y el movimiento, Phasmophobia muestra una hibridación increible entre la voz humana y la inteligencia artificial algorítmica.
Analizándolo con las "gafas de Manovich" podremos ver:
- La voz como dato. En el día a día, la voz es un medio de comunicación. En Phasmophobia, la voz se hibrida con el software de reconocimiento de voz como un parámetro de entrada, "escuchando constantemente la comunicación entre los jugadores, para asi extraer metadatos. Estos metadatos son utilizados en contra de los jugadores para modificar el comportamiento de la IA (el fantasma).
- Técnicas independientes del medio.
Este juego utiliza técnicas de procesamiento de datos para identificar palabras clave entre los jugadores o incluso el volumen que se utiliza (se entiende que cuando un jugador habla alto o pega un chillido, el pánico del jugador ha aumentado). Estas técnicas fusionan el lenguaje hablado con la lógica condicional del software.
- Hibridación de la presencia
Se crea una interacción donde la emoción real (el miedo, que se expresa a través de la voz) es captada, digitalizada y devuelta al usuario como una respuesta algorítmica, siendo una hibridación de la performance teatral con la interacción del humano-software. La finalidad de este juego es la creacion de un híbrido en el que el comportamiento humano es el código fuente, siendo cada partida jugada única e irrepetible
